HugoとGitHub Pagesを使ってます

Blog作成手順 🔗

  1. Hugoのインストール HugoをChocolateyを使ってインストール
    choco install hugo

  2. Hugoのプロジェクト作成(プロジェクトディレクトリを作成)
    hugo new site <プロジェクトディレクトリ>

  3. テーマの導入 Hugo Themesから好きなテンプレートを選んでください
    このページではminiを使ってます
    以下miniをテーマとして利用することを想定して進めますので, 適宜読み替えてください
    git submoduleで追加

cd <プロジェクトディレクトリ>
git init
git submodule add https://github.com/nodejh/hugo-theme-mini.git themes/mini

  1. ページの作成
    テーマ内のexampleSiteディレクトリをプロジェクトディレクトリにコピー
    hugo.tomlに以下を追加
    theme="mini"

  2. 起動
    ローカルでテストサーバーを起動
    hugo server
    http://localhost:1313で結果を確認できます

  3. サイトの設定
    hugo.tomlでサイトの設定をする

baseURL = "https://example.com/" # 自分のサイトのURL
languageCode = "ja" # 言語設定
title = "My New Hugo Site" # サイトのタイトル
theme = "mini" # 利用しているテーマ
  1. GitHub Pagesで公開
    publicディレクトリにプロジェクトを出力
    hugo
    publicディレクトリをリポジトリにpush

新しいコンテンツの作成方法 🔗

新しいページの作成
hugo new content content/posts/my-first-post.md
